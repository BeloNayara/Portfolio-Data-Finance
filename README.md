Bem-vindo ao meu repositório central de projetos. Aqui conecto o rigor das **Ciências Contábeis (CRC)** com o poder analítico do **Data Science (Unicamp)** para resolver problemas complexos do mercado financeiro e controladoria.


Projetos em Destaque

[Inteligência Financeira e Auditoria Digital (TCC)](./Auditoria_Bancaria)
Desenvolvimento de um modelo analítico para avaliação de riscos e consistência de demonstrações contábeis.
* **O Problema:** Identificação manual de inconsistências em balanços é lenta e sujeita a falhas.
* **A Solução:** Algoritmos em **Python** que aplicam testes estatísticos para validar a integridade dos dados financeiros.
* **Tecnologias:** Python, Pandas, Estatística Aplicada, Regras BACEN.

[Análise Macroeconômica Aplicada - Setor de Alimentos (BRF)](./Analise_BRF)
Integração de indicadores macroeconômicos com dados financeiros corporativos.
* **O Problema:** Necessidade de entender como variáveis externas (Inflação, Juros, Câmbio) impactam o bottom-line.
* **A Solução:** Dashboards analíticos e modelagem em **Python e R** para suporte à análise estratégica.
* **Tecnologias:** Python, R, Matplotlib/Seaborn, Análise de Séries Temporais.

[Modelagem de Risco e SQL Toolkit](./SQL_Financeiro)
Scripts avançados para extração de dados e análise de provisões (PELD) em carteiras de crédito.
* **Tecnologias:** SQL Server, PostgreSQL, Data Cleaning.
